# Installation

## Create the environment

```
npx create-nx-workspace@latest
? Workspace name (e.g., org name)     monorepo-workspace
? What to create in the new workspace react             [a workspace with a single React a
pplication]
? Application name                    nxreact
? Default stylesheet format           CSS
? Use Nx Cloud? (It's free and doesn't require registration.) No
```

```
nx g @nrwl/react:lib components
```

## Create library 'components' and set up storybook

```
yarn add @nrwl/storybook

yarn nx g storybook-configuration components
? Configure a cypress e2e app to run against the storybook instance? Yes
? Automatically generate story files for components declared in this library? Yes
? Automatically generate *.spec.ts files in the cypress e2e app generated by the cypress-configure generator? Yes

nx storybook components
```
